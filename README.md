# AppScreen Generator

Professional mockup generator for App Store and Google Play screenshots.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`

2. Run the app:
   `npm run dev`

## Features

- Multiple device frames (iPhone, iPad, Android)
- Custom backgrounds (gradients, patterns, images)
- Text overlays with customization
- Export to multiple sizes
- GIF animation export
